# Quantum-vs-Classical-Wave-Simulation-Schr-dinger-vs-Maxwell-
This project compares the behavior of quantum and classical waves using numerical simulation.  
Quantum system: 
Time-dependent Schrödinger equation 
Classical system: 
Wave equation (Maxwell analog)  
The goal is to demonstrate:  Wave packet dispersion in quantum mechanics Stable propagation in classical physics Relevance to nanoscale systems
